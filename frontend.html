HTML Última atualização: 24/11/2021

Definição 1.1

(Linguagem de Marcação de HiperTexto). Como a própria sigla já diz, HTML é uma linguagem de marcação (Não programação), é o bloco de construção da Web. Leia: https://developer.mozilla.org/pt-BR/docs/Web/HTML

Sintaxe 1.2

A sintaxe do HTML é bem simples, sendo constituída de “tags”.

Um elemento HTML sempre possui uma tag de abertura: <div>

E geralmente outra de fechamento: </div>

Dentro da tag HTML pode-se ter a definição de alguns atributos, exemplo: <input type="email">

CSS Última atualização: 24/11/2021

Definição 1.1

É uma linguagem de estilos (Também não é uma linguagem de programação) usada para descrever como os elementos do documentos serão apresentados (Cor, tamanho, margem, posição, animação etc...). Leia: https://developer.mozilla.org/pt-BR/docs/Web/CSS

O CSS é definido dentro da tag <style>, em um documento separado ou como um atributo style=”” de um elemento.
